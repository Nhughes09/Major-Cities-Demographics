# Global City Demographics Dashboard

Interactive dashboard showing projected ethnic demographic shifts for 20 major global cities from 2025 to 2100.

## Features

- **20 Cities** across 6 continents
- **Interactive Timeline** — slide from 2025 to 2100 in 5-year intervals
- **Multiple Chart Types** — stacked area, bar, and doughnut views
- **City Comparison** — compare up to 4 cities side-by-side
- **Regional Overview** — continent-level demographic summaries
- **Responsive Design** — works on mobile, tablet, and desktop

## Data Sources

Projections are modeled from observable macro-trends documented by:

- UN World Population Prospects 2022
- UN World Urbanization Prospects 2025
- Pew Research Center
- Eurostat
- US Census Bureau

## Deployment

Static site — deploy directly to Cloudflare Pages, Netlify, or any static host.

```bash
# Local preview
npx serve .
```
